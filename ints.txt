Добавление изменений и создание коммита:

Используйте команду git add <файлы> для добавления измененных файлов в индекс. Замените <файлы> именами файлов, которые вы хотите добавить.
Затем, используйте команду git commit -m "Описание коммита", чтобы создать коммит с описанием изменений.
Отправка изменений на удаленный репозиторий:

Используйте команду git push origin <ветка> для отправки ваших изменений на удаленный репозиторий. Замените <ветка> именем ветки, на которую вы хотите отправить изменения (обычно это master).
Проверка изменений на удаленном репозитории:

Перейдите на страницу вашего удаленного репозитория на GitHub, обновите страницу и проверьте, что ваши изменения успешно загружены.
